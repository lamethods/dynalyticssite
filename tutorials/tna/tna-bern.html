TNA Bern Workshop Resources

A curated hub of R packages, Python ports, interactive apps, hands-on tutorials, and references for the Bern Transition Network Analysis workshop.

Bern · Workshop Resources

Transition Network Analysis (TNA) models sequences of events as weighted directed networks, making it possible to uncover the temporal dynamics, transitions, and relational patterns that drive behavioral, psychological, and educational processes. The tna R package brings the method to practice with four model families — transition (Markov), frequency-based, co-occurrence, and attention-weighted — together with bootstrap stability, permutation-based group comparisons, community detection, and rich visualisation. Nestimate wraps and extends this stack into a unified estimation framework that also covers psychological networks and panel data. This page collects the packages, interactive Shiny and jamovi apps, hands-on tutorials, Nestimate articles, blog posts, and the published papers that ground the method — so you can move from raw event logs to validated network models in one place.

tna R Package

The reference R implementation of Transition Network Analysis. In addition to the tna R package, you can also try our Shiny app — an interactive web application for exploring sequences, communities, centrality, and network visualisation — and our Jamovi plugin, which brings TNA into the jamovi point-and-click environment with built-in statistical validation.

Nestimate R Package

Unified network estimation, analysis, and validation for behavioral, psychological, and panel data.

https://saqr.me/Nestimate/

A single interface for transition networks (transition, frequency, co-occurrence, attention-weighted), psychological networks, and panel-data networks — with bootstrap, simplicial-complex, and model-assessment tools layered on top.

Related packages

Sibling R packages and the Python port that round out the TNA ecosystem.

  • cograph · R Tools for the analysis, visualisation, and manipulation of dynamical, social, and complex networks — supports heterogeneous, multi-layer, and hierarchical networks with a simple syntax. Used throughout Nestimate’s visualisation pipeline.
    CRAN Docs GitHub
  • codyna · R Analysis of complex dynamic systems — the temporal unfolding of patterns, changes, and state transitions in behavioural data. Complements TNA with sequence-level pattern detection and outcome association.
    CRAN GitHub
  • tnapy · Python Full Python port of the TNA framework with numerical equivalence to the R package. Supports eight model types (relative, frequency, co-occurrence, attention-weighted, n-gram, gap-weighted…), network plots, heatmaps, and centrality charts — installable with pip.
    GitHub
